# zig-plscommit

A simple CLI tool to commit changes to git using zig.

## Installation

```bash
git clone https://github.com/davidprokopec/zig-plscommit.git
cd zig-plscommit
make build
```

### Using Zig

```bash
zig build
```

## Usage

```bash
plscommit <path>
```
